\# OMR Evaluation System



A Python + Streamlit web app to \*\*evaluate OMR sheets or CSV answer files\*\* and generate score reports for students.  



\---



\## \*\*Features\*\*



\- Upload an \*\*Answer Key CSV\*\*  

\- Upload \*\*Student Answers CSV\*\*  

\- Automatically \*\*calculate scores\*\* for each student  

\- Generate a \*\*summary report\*\* with:

&#x20; - Total students

&#x20; - Average score

&#x20; - Highest score

&#x20; - Lowest score

&#x20; - Total questions  

\- Download \*\*results\*\* and \*\*analysis report\*\* as CSV  



\---



\## \*\*Tech Used\*\*



\- Python 3.x  

\- \[Streamlit](https://streamlit.io/) – for web interface  

\- \[Pandas](https://pandas.pydata.org/) – for data handling  

\- NumPy – for calculations
